# Instruction :warning:
 
If you would like to download the code and try it for yourself:
 
- Clone the repository
- Install packages: `npm install`
- Install dependencies: `npm i [dependencies_name]`
- Launch: `npm start`
- Visit in your browser at: `http://localhost:8000`
